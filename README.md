## Project-Panavia
> Document biased roulette wheels and visualize data with Plotly and Dash. Spring 2021.

### Installation
```
pip3 install dash
pip3 install plotly
pip3 install ipinfo
pip3 install dash_table
pip3 install dash_core_components
pip3 install dash html_components

cd
git clone https://github.com/lxRbckl/Project-Panavia.git
```
